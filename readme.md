## 自行建置使用方式：
- 事先準備
  - 去discord developers裡面建立機器人
  - 下載最新釋出的版本壓縮檔
  - **解壓縮到一個空的資料夾** (**非常重要，這個資料夾會用來儲存所有log及其他必要檔案**)
- 環境建構
    - 安裝 **Python 3.10 或以上**
    - 安裝適合自己的IDE (推薦 **Visual Studio Code**)
    - 必要套件安裝：
      - 在終端機裡面輸入：
        - pip install discord
        - pip install python-dotenv
        - pip install py-cord
        - pip install pycord
        - pip install request
- 複製金鑰 (token) 貼在 **.env** 裡面，覆蓋 **bot token here**
- 在IDE內執行HydrodynamicX
